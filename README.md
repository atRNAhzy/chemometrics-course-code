## 化学自动化

对照课程实验顺序整理在 [化学自动化](化学自动化) 目录下，子目录名即实验编号。

- **01 串口通信与注射泵联动**：注射泵指令控制、继电器通断、数码管日期时间轮播、注射泵-天平自动校正。
- **02 传感器感知**：激光测距、超声-温度联动校正、pH 模块读取以及 Arduino 串口绘图集成。
- **03 闭环运动控制**：基于 Arduino Uno+CNC Shield 的直线模组，涵盖限位触发反向、霍尔传感器回零、AB 编码器高精度反馈。
- **05 三轴平台与移液**：多线程 UI 与串口通信解耦、状态锁保证指令一致性、阵列化定量移液流程和超时重传策略。
- **06 MG400 四轴机械臂**：搭载气动吸盘完成实验器材抓取与转运任务。
- **07 流动化学**：微流控平台进行靛蓝胭脂红氧化还原实验，结合图像处理提取色段并计算表观活化能。
- **08 AGV 与视觉识别**：AGV 小车与六轴机械臂协同，实现自动抓取与转运。

## 化学测量学综合实验

文件位于 [化学测量学综合实验](化学测量学综合实验) 目录，分为单分子器件测量与“光、机、电、通信”综合实验两部分。

- **单分子器件电学特性**：`单分子器件电学特性测量/简洁聚类.ipynb` 记录单分子器件测量数据的聚类与特征提取流程。
- **光机电实验平台**：`光机电/` 包含 `environment.yml`、`requirements.txt`、步进电机控制固件 `stepper&cond.ino`、以及 Python 源码 `src/`。实时采集的原始数据存放于 `results/raw/`，处理后的滴定与分析文本位于 `results/processed/`。
- **实验内容简介**：基于 Arduino Uno 与 CNC Shield 驱动步进电机实现电导滴定，配套 UI 与通信逻辑支持电机实时控制、数据采集、分析与可视化。

## 人工智能化学分析

相关作业放在 [人工智能化学分析](人工智能化学分析) 目录。

- `README.md`：记录学生信息与作业列表（第一周分子式不饱和度计算、第二周根据关键词下载 Elsevier 文献）。
- `week01/`：`dou_calculator.py` 计算有机物不饱和度。
- `week02/`：包含 `main.py` 及 `src/` 模块（`client.py`, `downloader.py`, `export.py` 等），支持根据配置自动检索并导出文献；`requirements.txt` 定义依赖，`outputs/` 保存关键词摘要与 PDF。
